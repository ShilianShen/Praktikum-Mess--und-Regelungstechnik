# 制作实验报告

一共有四份实验报告.

每个实验报告, 由以下部分构成:

- Motivation / Einleitung
- Versuchsaufbau
- Versuchsdurchführung
- Ergebnisdiskussion
- Fehlerdiskussion

设定的章节是:

1. Motivation / Einleitung 
2. Versuchsaufbau / Versuchsdurchführung
3. Ergebnisdiskussion
4. Fehlerdiskussion
5. Zusammenfassung
6. Referenze

报告原则上「不安排修改后重新提交」, 所以需要审核.

目前安排有以下分工:

- 实验数据
  - 图片
  - 代码
  - 纯数据
  - 审核
- 实验文本
  - 写手
  - 审核
- 论文整合
  - 编辑
  - 公式
  - 格式
  - 审核
- 提交
  - 执行
  - 审核
