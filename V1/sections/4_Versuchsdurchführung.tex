借助 MATLAB/Simulink 生成基本传递环节的阶跃响应。
仿真采用参数 $K_P = 2$、$T_1 = \SI{0.5}{\second}$ 和 $T_2 = \SI{1}{\second}$。

为了能够直接且清晰地比较不同系统的动态行为，所有输出信号通过 Mux 模块汇总，并在同一个 Scope（示波器）中显示。
首先使用单位阶跃信号作为输入，记录 P、I、PT1 和 PT2 环节的阶跃响应。
随后将输入信号改为正弦信号，并比较输入与输出之间的幅值变化和相位偏移。
该步骤用于验证线性时不变系统在正弦稳态下保持输入频率不变、只改变幅值和相位的性质。

随后在 MATLAB 中定义不同参数的 PT2 传递函数。
首先保持固有频率 $\omega_n$ 不变，并改变阻尼系数 $D$。
然后保持阻尼系数 $D$ 不变，并改变固有频率 $\omega_n$。
对每组参数分别绘制阶跃响应、Bode 图和零极点图。
通过这些图形比较响应速度、超调、振荡衰减、频率特性和极点位置之间的关系。

第一组传递函数的分母中二次项系数均为 $\frac{1}{3}$，因此 $T=\sqrt{\frac{1}{3}}$ 且 $\omega_n=\sqrt{3}$ 保持不变。
线性项系数不同，对应阻尼系数依次为
\[
  D = \frac{1}{3},\quad \frac{2}{3},\quad 1,\quad \frac{3}{2}.
\]
这组仿真主要用于观察阻尼从欠阻尼、临界阻尼到过阻尼时的变化。

第二组传递函数保持阻尼系数 $D=\frac{1}{3}$ 不变，改变二次项系数。
由此得到固有频率
\[
  \omega_n = \sqrt{3},\quad 2,\quad \sqrt{6}.
\]
这组仿真主要用于观察固有频率对响应速度、振荡频率和 Bode 图拐点位置的影响。

MATLAB 中使用 \texttt{legend} 标注各系统曲线，并使用 \texttt{exportgraphics} 将图形保存为 PDF 格式，供报告中引用。
